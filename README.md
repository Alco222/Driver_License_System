# Driver_License_System
Description
This is a desktop application built to simulate the government driver license management process. It allows administrators to manage appointments, process applications, and issue licenses.

## Features
* **Appointment Booking:** Users can book dates for exams.
* **Status Tracking:** Check if the license is approved or pending.
* **Admin Dashboard:** Manage all requests efficiently.

## Technologies Used
* **Language:** C#
* **Database:** Microsoft SQL Server
* **IDE:** Visual Studio

## How to Run
1. Clone the repository.
2. Import the database script (included) to SQL Server.
3. Run the solution file (.sln) in Visual Studio.
